# Vinyl Dance Battle

Веб-приложение для анонимных голосований на танцевальных батлах.

## Технологии

- Next.js 14 (App Router)
- Prisma + SQLite
- Tailwind CSS
- Lucide React
- React Server Actions

## Установка и запуск

### 1. Установить зависимости

```bash
npm install
```

### 2. Настроить .env

Создайте файл `.env` (можно скопировать `.env.example`):

```
ADMIN_PIN=1234
```

### 3. Инициализировать базу данных

```bash
npm run db:push
```

Это создаст/обновит файл `prisma/dev.db` с таблицами Poll и Option. При изменении схемы (например, добавлении полей) выполните команду снова.

### 4. Разработка

```bash
npm run dev
```

Приложение будет доступно на http://localhost:3000

### 5. Продакшн (build и start)

```bash
npm run build
npm start
```

Для продакшн-среды рекомендуется:

1. Указать `DATABASE_URL` в `.env`, если используете внешнюю SQLite-базу:
   ```
   DATABASE_URL="file:./prod.db"
   ```

2. Установить надёжный `ADMIN_PIN`.

3. Запускать через PM2 или systemd для стабильной работы.

## Функционал

- **Публичная лента:** просмотр опросов, голосование одним кликом
- **Защита от накрутки:** cookie хранит ID опросов, по которым уже проголосовали
- **Админ-панель:** `/admin` — ввод PIN → форма создания опроса (вопрос + 2–10 вариантов ответа)
